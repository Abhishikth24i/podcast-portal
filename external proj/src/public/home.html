How it works

1) Upload

Send audio directly into GridFS. We validate size and file type and store metadata like title and description.

2) Stream

Playback supports HTTP Range so listeners can scrub and resume quickly.

3) Manage

Edit titles/descriptions, list your library, and download or delete episodes.

4) Scale

Backed by MongoDB for reliability and growth from first episode to thousands.
